# angular-rxjs-lesson1
# Observable
Any stream of data

# Observer
  - Observes the stream
  - Methods to process notifications from the stream: next(), error(), complete()

# Subscriber
An observer that can unsubscribe

# Subscription
  - Represents the execution of an Observable
  - subscribe() returns a Subscription
